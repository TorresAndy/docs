# CryptoLay Ghost

CryptoLay Ghost hides your real API endpoints behind encrypted, rotating paths. Prevent API abuse, reconnaissance, and unauthorized access with intelligent endpoint obfuscation.

## What is Ghost?

Ghost is an API protection service that obfuscates your real endpoints behind encrypted, rotating paths. It prevents attackers from mapping your API structure and stops unauthorized access attempts.

## How It Works

### 1. Create a Manifest

Define your endpoint mappings and rotation rules:

```json
{
  "name": "API Protection Manifest",
  "endpoints": {
    "/api/v1/users/login": {
      "alias": "/e9k1zv",
      "rotation": "standard"
    },
    "/api/v1/payments/process": {
      "alias": "/x7m2qw",
      "rotation": "aggressive"
    }
  }
}
```

### 2. Generate Rotating Paths

Ghost automatically generates encrypted paths for your endpoints:

```javascript
// Real endpoint
POST /api/v1/users/login

// Ghost proxy endpoint
POST /e9k1zv

// Auto-rotated after 24h
POST /x7m2qw
```

### 3. Route Traffic

Proxy forwards requests to your real endpoints:

```javascript
// Client makes request to Ghost endpoint
const response = await fetch('https://ghost.cryptolay.com.br/e9k1zv', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'password123'
  })
});

// Ghost forwards to your real endpoint
// POST /api/v1/users/login
```

### 4. Monitor & Analyze

Track usage, detect abuse, and optimize performance:

```javascript
// Get metrics for your endpoints
const metrics = await fetch('https://api.cryptolay.com.br/api/ghost/metrics', {
  headers: {
    'X-API-Key': 'your_api_key'
  }
});
```

## Rotation Policies

### Standard Rotation
- **Frequency**: Every 24 hours
- **Use Case**: General API protection
- **Overhead**: Low
- **Protection**: Basic

### Aggressive Rotation
- **Frequency**: Every 6 hours
- **Use Case**: High-security applications
- **Overhead**: Medium
- **Protection**: Enhanced

### Super Aggressive Rotation
- **Frequency**: Every hour
- **Use Case**: Maximum security requirements
- **Overhead**: Higher
- **Protection**: Maximum
