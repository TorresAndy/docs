## Billing & Usage

### Usage Tracking

Monitor your usage across both products:

```javascript
// Get current usage
const usage = await fetch('https://api.cryptolay.com.br/api/billing/usage', {
  headers: {
    'X-API-Key': 'your_api_key'
  }
});

// Response includes both Shield and Ghost usage
{
  "shield": {
    "encryptions": 1250,
    "limit": 5000,
    "remaining": 3750
  },
  "ghost": {
    "requests": 850,
    "limit": 10000,
    "remaining": 9150
  }
}
```

### Overages

When you exceed your plan limits:

- **Free Tier**: Requests are blocked until next billing cycle
- **Pro Tier**: Pay per additional request/encryption
- **Enterprise**: Unlimited usage included

### Billing Cycles

- **Monthly**: Standard billing cycle
- **Annual**: 20% discount for annual prepayment
- **Custom**: Enterprise customers can negotiate custom terms

## Upgrading Plans

### Upgrade Process

1. **Choose New Plan**: Select the plan that fits your needs
2. **Update Billing**: Update payment information if needed
3. **Immediate Access**: New limits take effect immediately
4. **Prorated Billing**: Only pay for the difference

### Downgrading

- Can downgrade at any time
- Changes take effect at next billing cycle
- Unused credits are applied to final bill
