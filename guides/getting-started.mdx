# Getting Started with CryptoLay

<div className="text-center mb-8">
  <img src="/logo/cryptolay_icon.png" alt="CryptoLay Icon" className="h-12 mx-auto mb-4" />
</div>

This guide will walk you through integrating CryptoLay into your application, from initial setup to production deployment.

## Prerequisites

Before you begin, ensure you have:

- A CryptoLay account
- An API key with appropriate permissions
- Basic knowledge of REST APIs
- Your application ready for integration

## Step 1: Create Your Account

1. Visit [cryptolay.com.br](https://cryptolay.com.br)
2. Sign up for a free account
3. Verify your email address
4. Complete your profile setup

## Step 2: Create Your First API Key

```bash
curl -X POST https://api.cryptolay.com.br/api/customer/api-keys \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{
    "name": "Production API Key",
    "permissions": ["encrypt", "decrypt"]
  }'
```

## Step 3: Create a Manifest

A manifest defines which fields should be encrypted:

```bash
curl -X POST https://api.cryptolay.com.br/api/customer/manifests \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "name": "User Data Manifest",
    "fields": ["email", "phone", "ssn"],
    "description": "Encrypts sensitive user information"
  }'
```

## Step 4: Encrypt Your First Data

```javascript
const response = await fetch('https://api.cryptolay.com.br/api/encrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: {
      email: 'user@example.com',
      phone: '+1234567890'
    }
  })
});

const result = await response.json();
console.log(result.encrypted_data);
```

## Step 5: Decrypt Your Data

```javascript
const response = await fetch('https://api.cryptolay.com.br/api/decrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: {
      email: 'encrypted_email_value'
    },
    meta: {
      identifier: 'request_uuid',
      encrypted_key: 'kms_encrypted_aes_key',
      key_fingerprint: 'kms_key_id',
      timestamp: '2024-01-01T00:00:00Z',
      algorithm: 'AES-256-GCM'
    }
  })
});

const result = await response.json();
console.log(result.decrypted_data);
```

## SDK Integration

### JavaScript/Node.js

```bash
npm install cryptolay
```

```javascript
const CryptoLay = require('cryptolay');

const client = new CryptoLay({
  apiKey: process.env.CRYPTOLAY_API_KEY
});

// Encrypt data
const encrypted = await client.encrypt({
  manifestId: 'manifest-uuid',
  data: { email: 'user@example.com' }
});

// Decrypt data
const decrypted = await client.decrypt({
  manifestId: 'manifest-uuid',
  data: encrypted.data,
  meta: encrypted.meta
});
```

### Python

```bash
pip install cryptolay
```

```python
import cryptolay

client = cryptolay.Client(api_key=os.environ['CRYPTOLAY_API_KEY'])

# Encrypt data
encrypted = client.encrypt(
    manifest_id='manifest-uuid',
    data={'email': 'user@example.com'}
)

# Decrypt data
decrypted = client.decrypt(
    manifest_id='manifest-uuid',
    data=encrypted['data'],
    meta=encrypted['meta']
)
```

## Best Practices

### Error Handling

```javascript
try {
  const result = await client.encrypt({
    manifestId: 'manifest-uuid',
    data: { email: 'user@example.com' }
  });
} catch (error) {
  if (error.code === 401) {
    console.error('Invalid API key');
  } else if (error.code === 400) {
    console.error('Invalid request data');
  } else {
    console.error('Encryption failed:', error.message);
  }
}
```

### Rate Limiting

```javascript
// Implement exponential backoff
const encryptWithRetry = async (data, maxRetries = 3) => {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await client.encrypt(data);
    } catch (error) {
      if (error.code === 429 && i < maxRetries - 1) {
        await new Promise(resolve => 
          setTimeout(resolve, Math.pow(2, i) * 1000)
        );
        continue;
      }
      throw error;
    }
  }
};
```
