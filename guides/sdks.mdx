# CryptoLay SDKs

Official SDKs for integrating CryptoLay into your applications. Choose the SDK that matches your programming language.

## JavaScript/Node.js SDK

### Installation

```bash
npm install cryptolay
```

### Basic Usage

```javascript
const CryptoLay = require('cryptolay');

const client = new CryptoLay({
  apiKey: process.env.CRYPTOLAY_API_KEY
});

// Encrypt data
const encrypted = await client.encrypt({
  manifestId: 'manifest-uuid',
  data: { email: 'user@example.com' }
});

// Decrypt data
const decrypted = await client.decrypt({
  manifestId: 'manifest-uuid',
  data: encrypted.data,
  meta: encrypted.meta
});
```

### Advanced Configuration

```javascript
const client = new CryptoLay({
  apiKey: process.env.CRYPTOLAY_API_KEY,
  baseURL: 'https://api.cryptolay.com',
  timeout: 30000,
  retries: 3
});
```

### Error Handling

```javascript
try {
  const result = await client.encrypt({
    manifestId: 'manifest-uuid',
    data: { email: 'user@example.com' }
  });
} catch (error) {
  if (error.code === 401) {
    console.error('Invalid API key');
  } else if (error.code === 400) {
    console.error('Invalid request data');
  } else {
    console.error('Encryption failed:', error.message);
  }
}
```

## Python SDK

### Installation

```bash
pip install cryptolay
```

### Basic Usage

```python
import cryptolay
import os

client = cryptolay.Client(api_key=os.environ['CRYPTOLAY_API_KEY'])

# Encrypt data
encrypted = client.encrypt(
    manifest_id='manifest-uuid',
    data={'email': 'user@example.com'}
)

# Decrypt data
decrypted = client.decrypt(
    manifest_id='manifest-uuid',
    data=encrypted['data'],
    meta=encrypted['meta']
)
```

### Advanced Configuration

```python
import cryptolay

client = cryptolay.Client(
    api_key=os.environ['CRYPTOLAY_API_KEY'],
    base_url='https://api.cryptolay.com',
    timeout=30,
    retries=3
)
```

### Error Handling

```python
try:
    result = client.encrypt(
        manifest_id='manifest-uuid',
        data={'email': 'user@example.com'}
    )
except cryptolay.AuthenticationError:
    print("Invalid API key")
except cryptolay.ValidationError:
    print("Invalid request data")
except cryptolay.APIError as e:
    print(f"API error: {e.message}")
```

## Ruby SDK

### Installation

```bash
gem install cryptolay
```

### Basic Usage

```ruby
require 'cryptolay'

client = Cryptolay::Client.new(api_key: ENV['CRYPTOLAY_API_KEY'])

# Encrypt data
encrypted = client.encrypt(
  manifest_id: 'manifest-uuid',
  data: { email: 'user@example.com' }
)

# Decrypt data
decrypted = client.decrypt(
  manifest_id: 'manifest-uuid',
  data: encrypted[:data],
  meta: encrypted[:meta]
)
```

### Advanced Configuration

```ruby
client = Cryptolay::Client.new(
  api_key: ENV['CRYPTOLAY_API_KEY'],
  base_url: 'https://api.cryptolay.com',
  timeout: 30,
  retries: 3
)
```

### Error Handling

```ruby
begin
  result = client.encrypt(
    manifest_id: 'manifest-uuid',
    data: { email: 'user@example.com' }
  )
rescue Cryptolay::AuthenticationError
  puts "Invalid API key"
rescue Cryptolay::ValidationError
  puts "Invalid request data"
rescue Cryptolay::APIError => e
  puts "API error: #{e.message}"
end
```

## Go SDK

### Installation

```bash
go get github.com/cryptolay/go-cryptolay
```

### Basic Usage

```go
package main

import (
    "fmt"
    "os"
    "github.com/cryptolay/go-cryptolay"
)

func main() {
    client := cryptolay.NewClient(os.Getenv("CRYPTOLAY_API_KEY"))

    // Encrypt data
    encrypted, err := client.Encrypt(cryptolay.EncryptRequest{
        ManifestID: "manifest-uuid",
        Data: map[string]interface{}{
            "email": "user@example.com",
        },
    })
    if err != nil {
        panic(err)
    }

    // Decrypt data
    decrypted, err := client.Decrypt(cryptolay.DecryptRequest{
        ManifestID: "manifest-uuid",
        Data:       encrypted.Data,
        Meta:       encrypted.Meta,
    })
    if err != nil {
        panic(err)
    }

    fmt.Printf("Decrypted: %+v\n", decrypted.Data)
}
```

## PHP SDK

### Installation

```bash
composer require cryptolay/cryptolay-php
```

### Basic Usage

```php
<?php

require 'vendor/autoload.php';

use Cryptolay\CryptolayClient;

$client = new CryptolayClient($_ENV['CRYPTOLAY_API_KEY']);

// Encrypt data
$encrypted = $client->encrypt([
    'manifest_id' => 'manifest-uuid',
    'data' => ['email' => 'user@example.com']
]);

// Decrypt data
$decrypted = $client->decrypt([
    'manifest_id' => 'manifest-uuid',
    'data' => $encrypted['data'],
    'meta' => $encrypted['meta']
]);
```

## Testing

### JavaScript/Node.js

```javascript
const CryptoLay = require('cryptolay');

// Mock client for testing
const mockClient = new CryptoLay({
  apiKey: 'test_key',
  baseURL: 'http://localhost:3000'
});

// Test encryption
const result = await mockClient.encrypt({
  manifestId: 'test-manifest',
  data: { email: 'test@example.com' }
});
```

### Python

```python
import cryptolay
from unittest.mock import patch

# Mock client for testing
with patch('cryptolay.Client') as mock_client:
    mock_client.return_value.encrypt.return_value = {
        'data': {'email': 'encrypted_value'},
        'meta': {'identifier': 'test_id'}
    }
    
    client = cryptolay.Client(api_key='test_key')
    result = client.encrypt(
        manifest_id='test-manifest',
        data={'email': 'test@example.com'}
    )
```

## Migration Guide

### From REST API to SDK

If you're currently using the REST API directly:

```javascript
// Before: Direct API calls
const response = await fetch('https://api.cryptolay.com/api/encrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: { email: 'user@example.com' }
  })
});

// After: SDK usage
const client = new CryptoLay({ apiKey: 'your_api_key' });
const result = await client.encrypt({
  manifestId: 'manifest-uuid',
  data: { email: 'user@example.com' }
});
```
