## How It Works

### Shield - Field-Level Encryption

#### 1. Create a Manifest

Define which fields should be encrypted in your data:

```json
{
  "name": "User Data Manifest",
  "fields": ["email", "phone", "ssn"],
  "description": "Encrypts sensitive user information"
}
```

#### 2. Encrypt Your Data

Send your JSON payload to Shield for encryption:

```javascript
const response = await fetch('https://api.cryptolay.com/api/encrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: {
      user_id: "12345",
      email: 'user@example.com',
      phone: '+1234567890',
      ssn: '123-45-6789'
    }
  })
});
```

#### 3. Store Encrypted Data

Store the encrypted data and metadata securely in your database.

#### 4. Decrypt When Needed

Retrieve and decrypt data when required:

```javascript
const decrypted = await fetch('https://api.cryptolay.com/api/decrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: encryptedData,
    meta: metadata
  })
});
```

### Ghost - API Protection

#### 1. Create Endpoint Mappings

Define your endpoint mappings and rotation rules:

```json
{
  "name": "API Protection Manifest",
  "endpoints": {
    "/api/v1/users/login": {
      "alias": "/e9k1zv",
      "rotation": "standard"
    }
  }
}
```

#### 2. Use Ghost Aliases

Clients use encrypted aliases instead of real endpoints:

```javascript
// Instead of: POST /api/v1/users/login
// Use: POST /e9k1zv

const response = await fetch('https://ghost.cryptolay.com.br/e9k1zv', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'password123'
  })
});
```

#### 3. Monitor & Analyze

Track usage, detect abuse, and optimize performance:

```javascript
const metrics = await fetch('https://api.cryptolay.com.br/api/ghost/metrics', {
  headers: {
    'X-API-Key': 'your_api_key'
  }
});
```

## Use Cases

### Healthcare (HIPAA Compliance)

Encrypt patient data to meet HIPAA requirements:

```javascript
// Encrypt patient information
const patientData = {
  name: "John Doe",
  ssn: "123-45-6789",
  diagnosis: "Hypertension"
};

const encrypted = await cryptolay.encrypt({
  manifestId: 'patient-data-manifest',
  data: patientData
});
```

### Financial Services (PCI-DSS)

Secure payment information:

```javascript
// Encrypt payment data
const paymentData = {
  cardNumber: "4111-1111-1111-1111",
  cvv: "123",
  expiryDate: "12/25"
};

const encrypted = await cryptolay.encrypt({
  manifestId: 'payment-data-manifest',
  data: paymentData
});
```

### E-commerce (GDPR Compliance)

Protect customer personal data:

```javascript
// Encrypt customer information
const customerData = {
  email: "customer@example.com",
  address: "123 Main St",
  phone: "+1234567890"
};

const encrypted = await cryptolay.encrypt({
  manifestId: 'customer-data-manifest',
  data: customerData
});
```
