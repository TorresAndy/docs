## What is CryptoLay?

CryptoLay provides a secure, scalable encryption service that allows you to encrypt sensitive data at the field level while maintaining compliance with industry standards. Our platform handles the complexity of key management, encryption algorithms, and audit trails so you can focus on building your application.

## How It Works

### 1. Create a Manifest

Define which fields should be encrypted in your data:

```json
{
  "name": "User Data Manifest",
  "fields": ["email", "phone", "ssn"],
  "description": "Encrypts sensitive user information"
}
```

### 2. Encrypt Your Data

Send your data to CryptoLay for encryption:

```javascript
const response = await fetch('https://api.cryptolay.com/api/encrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: {
      email: 'user@example.com',
      phone: '+1234567890'
    }
  })
});
```

### 3. Store Encrypted Data

Store the encrypted data and metadata securely in your database.

### 4. Decrypt When Needed

Retrieve and decrypt data when required:

```javascript
const decrypted = await fetch('https://api.cryptolay.com/api/decrypt', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    manifest_id: 'manifest-uuid',
    data: encryptedData,
    meta: metadata
  })
});
```
