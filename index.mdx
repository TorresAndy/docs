## What is CryptoLay?

CryptoLay provides a secure, scalable encryption service that allows you to encrypt sensitive data at the field level while maintaining compliance with industry standards. Our platform handles the complexity of key management, encryption algorithms, and audit trails so you can focus on building your application.

## Architecture

The CryptoLay platform consists of four main applications:

- **Main API** (Node.js Lambda) - Core encryption/decryption service
- **Customer Admin** (Next.js) - Customer dashboard for managing encryption manifests
- **System Admin** (Next.js) - Administrative interface for system management  
- **Landing Page** (Next.js) - Public marketing and signup page

## Getting Started

### Prerequisites

- Node.js 18+
- Docker & Docker Compose
- PostgreSQL
- AWS Account (for KMS and Lambda deployment)

### Quick Setup

1. **Start the database:**
   ```bash
   docker-compose up postgres -d
   ```

2. **Apply database schema:**
   ```bash
   psql -h localhost -U postgres -d cryptolay -f database/schema.sql
   ```

3. **Start the main API:**
   ```bash
   cd main-api
   npm install
   npm run dev
   ```

4. **Start frontend applications:**
   ```bash
   # Customer Admin
   cd customer-admin
   npm install
   npm run dev

   # System Admin  
   cd system-admin
   npm install
   npm run dev

   # Landing Page
   cd landing-page
   npm install
   npm run dev
   ```
