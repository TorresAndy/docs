# Ghost Overview

CryptoLay Ghost hides your real API endpoints behind encrypted, rotating paths. Prevent API abuse, reconnaissance, and unauthorized access with intelligent endpoint obfuscation.

## What is Ghost?

Ghost is an API protection service that obfuscates your real endpoints behind encrypted, rotating paths. It prevents attackers from mapping your API structure and stops unauthorized access attempts.

## How It Works

### 1. Create a Manifest

Define your endpoint mappings and rotation rules:

```json
{
  "name": "API Protection Manifest",
  "description": "Protects API endpoints from abuse and reconnaissance",
  "endpoints": {
    "/api/v1/users/login": {
      "alias": "/e9k1zv",
      "rotation": "standard",
      "rateLimit": {
        "requests": 100,
        "window": "1m"
      }
    },
    "/api/v1/payments/process": {
      "alias": "/x7m2qw",
      "rotation": "aggressive",
      "rateLimit": {
        "requests": 50,
        "window": "1m"
      }
    },
    "/api/v1/admin/users": {
      "alias": "/y8n3rx",
      "rotation": "super-aggressive",
      "rateLimit": {
        "requests": 10,
        "window": "1m"
      }
    }
  }
}
```

### 2. Generate Rotating Paths

Ghost automatically generates encrypted paths for your endpoints:

```javascript
// Real endpoints
POST /api/v1/users/login
POST /api/v1/payments/process
GET /api/v1/admin/users

// Ghost proxy endpoints (initial)
POST /e9k1zv
POST /x7m2qw
GET /y8n3rx

// After rotation (24h for standard)
POST /a1b2c3
POST /d4e5f6
GET /g7h8i9
```

### 3. Route Traffic

Proxy forwards requests to your real endpoints:

```javascript
// Client makes request to Ghost endpoint
const response = await fetch('https://ghost.cryptolay.com.br/e9k1zv', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your_api_key'
  },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'password123'
  })
});

// Ghost forwards to your real endpoint
// POST /api/v1/users/login
```

### 4. Monitor & Analyze

Track usage, detect abuse, and optimize performance:

```javascript
// Get metrics for your endpoints
const metrics = await fetch('https://api.cryptolay.com.br/api/ghost/metrics', {
  headers: {
    'X-API-Key': 'your_api_key'
  }
});

// Response includes traffic analysis
{
  "endpoints": {
    "/e9k1zv": {
      "requests": 1250,
      "errors": 12,
      "avgResponseTime": 45,
      "abuseAttempts": 3
    }
  }
}
```

## Rotation Policies

### Standard Rotation
- **Frequency**: Every 24 hours
- **Use Case**: General API protection
- **Overhead**: Low
- **Protection**: Basic
- **Best For**: Public APIs, general web services

### Aggressive Rotation
- **Frequency**: Every 6 hours
- **Use Case**: High-security applications
- **Overhead**: Medium
- **Protection**: Enhanced
- **Best For**: Financial services, healthcare APIs

### Super Aggressive Rotation
- **Frequency**: Every hour
- **Use Case**: Maximum security requirements
- **Overhead**: Higher
- **Protection**: Maximum
- **Best For**: Government, military, critical infrastructure

## Architecture

### Proxy Layer

Ghost operates as a reverse proxy that:
- Intercepts requests to obfuscated endpoints
- Validates API keys and rate limits
- Forwards requests to real endpoints
- Returns responses to clients

### Rotation Engine

The rotation engine:
- Generates encrypted paths using AES-256
- Rotates paths based on configured policies
- Maintains mapping between real and proxy endpoints
- Handles path transitions seamlessly

### Analytics Engine

The analytics engine provides:
- Real-time traffic monitoring
- Abuse pattern detection
- Performance metrics
- Geographic analysis
